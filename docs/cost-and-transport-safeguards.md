# 成本与交通安全策略

## AI 成本保护

- 北京时间每日重置：完整生成 2 次、整天修改 2 次、局部修改 5 次。
- 默认全站软预算 5 美元、硬预算 8 美元。软预算后关闭非必要 AI 质量修复；硬预算后拒绝新的付费 AI 请求，读取和分享不受影响。
- OpenAI Responses API 的 input、output、cached input、reasoning token 与估算成本写入 `analytics_events.metadata`；不保存完整 Prompt、Cookie 或 IP。
- 同一访客只允许一个完整生成任务；请求使用幂等键。IP 只在服务端生成不可逆短哈希用于小时级限流。
- 模型价格集中在 `src/server/ai/model-pricing.ts`，价格变更时应先核对供应商官方页面。

## 交通数据策略

当前没有接入地图、铁路或航班实时 API。常见城市对使用 `intercity-transport.ts` 的保守区间；未知城市对直接隐藏模块，不显示“待确认”占位内容。

若下一阶段接入，优先评估高德 Web 服务路径规划：它覆盖中国大陆驾车与公共交通，服务端 Key 可同时供网页和未来小程序后端复用。Key 只能保存在服务端环境变量；城市对与短期路线结果可按起终点、方式和日期缓存。预计一次生成增加 2～4 次请求（地理编码、去程、返程），不得在未确认账号资质、额度和费用前启用。

## 回滚

本轮未修改数据库表结构。关闭新保护可通过环境变量调整限额与预算；代码回滚不会影响既有 JSONB 攻略和历史事件。
